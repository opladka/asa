Introduction to MATLAB and its Basic Commands

Q01
What is MATLAB? What does the acronym stand for?
▶
Answer
MATLAB stands for MATrix LABoratory. It is a high-performance numerical computing environment and programming language developed by MathWorks. It integrates computation, visualization, and programming in an easy-to-use environment where problems and solutions are expressed in familiar mathematical notation. It is widely used for algorithm development, data analysis, simulation, and scientific computing.
Q02
What is the difference between a script and a function in MATLAB?
▶
Answer
A Script is a file containing a sequence of MATLAB commands. It operates on the base workspace and does not accept input arguments or return output values. A Function is also an M-file but it has its own local workspace, accepts input arguments, returns output values, and the variables created inside a function are not accessible outside it. Functions begin with the keyword function and must be saved with the same name as the function.
Q03
What is the purpose of clc, clear, and close all commands?
▶
Answer
clc — Clears the Command Window display (does not delete variables).
clear or clear all — Removes all variables from the workspace memory.
close all — Closes all open figure windows. These three commands are typically used together at the beginning of a program to ensure a clean environment for execution.
Q04
What is the difference between .* (dot multiply) and * (matrix multiply) in MATLAB?
▶
Answer
* performs matrix multiplication (following linear algebra rules, inner dimensions must match). .* performs element-wise multiplication (corresponding elements of two arrays are multiplied; arrays must be the same size). Similarly, ./ and .^ are element-wise division and power. Example: if A=[1 2; 3 4] and B=[5 6; 7 8], then A.*B = [5 12; 21 32] whereas A*B gives matrix product.
Q05
What is the difference between a semicolon (;) and a comma (,) in MATLAB?
▶
Answer
A semicolon (;) at the end of a statement suppresses the output (result is not displayed in the command window). Inside a matrix, it separates rows. A comma (,) separates elements in the same row of a matrix, or separates arguments in a function call. For example: A = [1,2; 3,4] creates a 2×2 matrix. Without the semicolon: x = 5 prints x = 5 on screen; with it, x = 5; stores silently.
Q06
How do you create a linearly spaced vector in MATLAB?
▶
Answer
Two methods:
Colon operator: x = start:step:end e.g., x = 0:0.1:10 creates a vector from 0 to 10 with step 0.1.
linspace: x = linspace(start, end, N) creates N evenly spaced points between start and end. For example, linspace(0, 2*pi, 100) gives 100 points from 0 to 2π.

EXPERIMENT 02 · SIGNAL ANALYSIS

Q01
Define the unit step function mathematically and state its Laplace transform.
▶
Answer
The unit step function u(t) is defined as:
u(t) = 1, for t ≥ 0   and   u(t) = 0, for t < 0
Its Laplace transform is: L{u(t)} = 1/s
Its Z-transform (discrete): Z{u[n]} = z/(z-1), |z| > 1
In MATLAB, the unit step can be implemented using the heaviside(t) function, or as u = (t >= 0) for vectors.
Q02
What is a unit impulse (Dirac delta) function? State its properties.
▶
Answer
The unit impulse δ(t) (Dirac delta) is defined such that:
δ(t) = 0, for t ≠ 0 and ∫δ(t)dt = 1 (over all time)
Properties:
  • Sifting: ∫f(t)δ(t-t₀)dt = f(t₀)
  • Laplace: L{δ(t)} = 1
  • It is the derivative of the unit step: δ(t) = du(t)/dt
  • Area = 1, duration → 0, amplitude → ∞
Q03
What is the relationship between unit step, unit impulse, and unit ramp signals?
▶
Answer
These three signals are related by integration and differentiation:
δ(t) = d/dt [u(t)] — Impulse is derivative of step
u(t) = d/dt [r(t)] — Step is derivative of ramp
r(t) = ∫u(t)dt — Ramp is integral of step
u(t) = ∫δ(t)dt — Step is integral of impulse
The ramp r(t) = t·u(t); its Laplace transform is 1/s². The parabolic signal p(t) = (t²/2)·u(t) with Laplace transform 1/s³.
Q04
Define energy and power signals. Which of the standard signals are energy signals and which are power signals?
▶
Answer
Energy Signal: A signal with finite, non-zero energy. Energy E = ∫|x(t)|² dt < ∞. Power P = 0.
Power Signal: A signal with finite, non-zero average power. Power P = lim(T→∞) (1/2T)∫|x(t)|² dt < ∞.
Classification:
  • Sinusoidal signal → Power signal (periodic, infinite duration)
  • Exponential decaying → Energy signal (finite energy)
  • Unit step, ramp → Neither strictly (infinite energy and infinite/zero power)
  • Rectangular pulse → Energy signal
Q05
What is a causal signal? Give examples.
▶
Answer
A signal x(t) is said to be causal if x(t) = 0 for all t < 0. Such signals exist only for t ≥ 0. Examples: unit step u(t), unit ramp r(t) = t·u(t), decaying exponential e^(-at)·u(t). An anti-causal signal exists for t < 0. A non-causal signal exists for both t < 0 and t ≥ 0. Physical systems produce causal outputs, making causal signals important in system analysis.
Q06
What is the difference between continuous-time and discrete-time signals?
▶
Answer
Continuous-time signals are defined for every instant of time (x(t), t ∈ ℝ). They can take any value at any time. Examples: sinusoidal signals from nature, analog voltage.
Discrete-time signals are defined only at specific (usually equally spaced) time instants (x[n], n ∈ ℤ). They are obtained by sampling continuous signals. In MATLAB, plot() is used for CT signals and stem() is used for DT signals. Sampling theorem (Nyquist) states sampling frequency must be ≥ 2 × max frequency.
Q07
What is a periodic signal? Define its period and frequency.
▶
Answer
A signal x(t) is periodic if there exists a positive T such that x(t+T) = x(t) for all t. The smallest such T is the fundamental period. Frequency f = 1/T Hz. Angular frequency ω = 2πf = 2π/T rad/s. For sinusoidal signals: x(t) = A·sin(ωt + φ) where A is amplitude, ω is angular frequency, φ is phase angle. Signals that don't repeat are aperiodic.
Q08
What are the properties of the exponential signal e^(at)?
▶
Answer
The exponential signal x(t) = A·e^(at):
  • If a < 0: Decaying exponential — diminishes over time. Important in natural system responses. Example: capacitor discharge.
  • If a > 0: Growing exponential — increases over time (unstable systems).
  • If a = 0: Constant signal.
  • If a = jω: Complex exponential e^(jωt) = cos(ωt) + j·sin(ωt) — relates to sinusoids via Euler's formula.
Laplace transform: L{e^(-at)u(t)} = 1/(s+a)

Answer
Unit Step:

Q10
What is even and odd decomposition of a signal?
▶
Answer
Any signal x(t) can be decomposed into:
Even component: x_e(t) = [x(t) + x(-t)] / 2 — symmetric about t=0, x_e(t) = x_e(-t)
Odd component: x_o(t) = [x(t) - x(-t)] / 2 — anti-symmetric, x_o(t) = -x_o(-t)
Such that x(t) = x_e(t) + x_o(t). The unit step function can be written as: u(t) = 1/2 + (1/2)·sgn(t) where the even part is 1/2 and odd part is (1/2)·sgn(t).
Q11
What is time shifting and time scaling of signals?
▶
Answer
Time Shifting: x(t - t₀) shifts the signal by t₀ to the right (delay); x(t + t₀) shifts left (advance).
Time Scaling: x(at) — if |a| > 1, signal is compressed; if 0 < |a| < 1, signal is expanded. If a < 0, signal is time-reversed as well.
Time Reversal: x(-t) flips the signal about the vertical axis (t=0).
In MATLAB, time shifting is achieved by replacing the time vector: y = x(t - t0)

EXPERIMENT 03 · SIGNAL PROCESSING
Plot the Linear Convolution of Two Sequences
10 Questions
+
Q01
Define linear convolution mathematically. What does it represent physically?
▶
Answer
The linear convolution of signals x(t) and h(t) is defined as:
y(t) = x(t) * h(t) = ∫ x(τ)·h(t-τ) dτ (continuous)
For discrete sequences: y[n] = Σ x[k]·h[n-k] (k from -∞ to ∞)
Physical significance: In an LTI (Linear Time-Invariant) system, convolution of the input signal x(t) with the system's impulse response h(t) gives the output y(t). It represents how the system processes the input over time, considering the system's memory.
Q02
If two sequences have lengths M and N, what is the length of their linear convolution?
▶
Answer
The length of the linear convolution of two sequences of lengths M and N is:
Length = M + N - 1
For example, if x has 5 samples and h has 3 samples, the output y will have 5 + 3 - 1 = 7 samples.
In MATLAB, the conv(x, h) function automatically gives the output of length M+N-1. The starting index of the output is the sum of starting indices of the two sequences.
Q03
State the properties of linear convolution.
▶
Answer
Properties of linear convolution:
1. Commutative: x * h = h * x
2. Associative: (x * h₁) * h₂ = x * (h₁ * h₂)
3. Distributive: x * (h₁ + h₂) = x*h₁ + x*h₂
4. Identity: x * δ = x (convolving with unit impulse returns original)
5. Shift: If y = x * h, then shifting x by n₀ shifts y by n₀
6. Width: Length of output = M + N - 1
Q04
What is the difference between linear convolution and circular convolution?
▶
Answer
Linear Convolution: Output length = M+N-1. Sequences are treated as aperiodic. Used in time-domain analysis of LTI systems. MATLAB: conv(x, h).
Circular Convolution: Output length = N (same as input length). Treats sequences as periodic with period N. Used in DFT/FFT computations. MATLAB: cconv(x, h, N) or via ifft(fft(x).*fft(h)).
Key fact: N-point circular convolution equals linear convolution only when N ≥ M+N-1 (i.e., sequences are zero-padded sufficiently to avoid aliasing).
Q05
How is convolution related to the Fourier/Laplace transform?
▶
Answer
The Convolution Theorem states that convolution in the time domain corresponds to multiplication in the frequency domain:
x(t) * h(t) ↔ X(ω)·H(ω) (Fourier Transform)
L{x(t) * h(t)} = X(s)·H(s) (Laplace Transform)
This is why the transfer function H(s) = Y(s)/X(s) is valid — it is the Laplace transform of the impulse response. This theorem makes frequency-domain analysis computationally efficient compared to direct convolution.
Q06
What is the impulse response of a system? Why is it important?
▶
Answer
The impulse response h(t) of a system is its output when the input is a unit impulse δ(t). It completely characterizes an LTI system. Importance:
  • The output for any input x(t) is: y(t) = x(t) * h(t)
  • If h(t) = 0 for t < 0, the system is causal
  • If ∫|h(t)|dt < ∞, the system is BIBO stable
  • Its Laplace transform gives the transfer function H(s)
Q07
What is deconvolution? Where is it used?
▶
Answer
Deconvolution is the inverse operation of convolution. Given y = x * h and knowing y and one of x or h, deconvolution recovers the other signal. In MATLAB: [q, r] = deconv(y, h). Applications:
  • Channel equalization in communications
  • Image restoration/deblurring
  • Seismic signal processing (recovering earth structure from reflected waves)
  • System identification (finding impulse response from input-output data)
Q08
What is an LTI system? State its properties.
▶
Answer
An LTI (Linear Time-Invariant) system satisfies two properties:
1. Linearity: Superposition holds — if x₁→y₁ and x₂→y₂, then (ax₁+bx₂)→(ay₁+by₂).
2. Time-Invariance: If x(t)→y(t), then x(t-t₀)→y(t-t₀). A delayed input gives a delayed but otherwise identical output.
LTI systems are fully characterized by their impulse response h(t) or equivalently by their transfer function H(s). Convolution is the fundamental operation relating input and output.
Q09
Explain BIBO stability of a system. Give the condition in terms of impulse response.
▶
Answer
BIBO (Bounded Input Bounded Output) Stability: A system is BIBO stable if every bounded input produces a bounded output. Mathematically, if |x(t)| ≤ M < ∞, then |y(t)| ≤ K < ∞.
Condition in terms of h(t): An LTI system is BIBO stable if and only if its impulse response is absolutely integrable:
∫|h(t)|dt < ∞
For discrete systems: Σ|h[n]| < ∞
In the s-domain: all poles of H(s) must lie in the left half of the s-plane (real parts negative).
Q10
Write the MATLAB code to compute and plot linear convolution of x = [1, 2, 3] and h = [1, 1, 1].
▶
Answer
x = [1, 2, 3];
h = [1, 1, 1];
y = conv(x, h);
n = 0 : length(y)-1;
stem(n, y);
title('Linear Convolution');
xlabel('n'); ylabel('y[n]'); grid on;
The result will be y = [1, 3, 6, 5, 3] (length = 3+3-1 = 5). Manual verification: convolve term by term using the flip-and-slide method.

EXPERIMENT 04 · RLC CIRCUITS
Transient Response of Series RLC Circuit — CRO & MATLAB Verification
12 Questions
+
Q01
What is the transient response of a circuit? How is it different from the steady-state response?
▶
Answer
Transient response is the portion of the circuit's response that decays to zero over time after a disturbance (such as switching). It arises due to energy stored in inductors and capacitors.
Steady-state response is the response remaining after the transient has died out — it persists indefinitely and depends on the input type.
The total response = Transient (natural/complementary) + Steady-state (forced/particular).
Transient duration depends on the time constants of the circuit.
Q02
Derive the characteristic equation of a series RLC circuit. What is the natural frequency and damping ratio?
▶
Answer
For a series RLC circuit with voltage v(t) applied, applying KVL:
L(d²i/dt²) + R(di/dt) + (1/C)i = dv/dt
Dividing by L, the characteristic equation in s-domain is:
s² + (R/L)s + 1/(LC) = 0
Natural frequency: ωₙ = 1/√(LC) rad/s
Damping ratio: ζ = R/(2)·√(C/L) = R/(2ωₙL)
Roots: s₁,₂ = -ζωₙ ± ωₙ√(ζ²-1)
Q03
Define overdamped, underdamped, and critically damped conditions. What are the waveforms observed on CRO?
▶
Answer
1. Overdamped (ζ > 1): Two distinct real negative roots. Response decays exponentially without oscillations. Slow to reach steady state. CRO shows: smooth exponential decay.
2. Critically Damped (ζ = 1): Two equal real negative roots. Fastest non-oscillatory decay. CRO shows: exponential decay, fastest approach to final value.
3. Underdamped (ζ < 1): Complex conjugate roots with negative real parts. Response oscillates while decaying. CRO shows: damped sinusoidal waveform with overshoot.
Undamped (ζ = 0): Pure imaginary roots, sustained oscillation.
Q04
What is resonance in a series RLC circuit? Define resonant frequency and Q factor.
▶
Answer
Resonance occurs when inductive reactance equals capacitive reactance: ωL = 1/(ωC).
Resonant frequency: ω₀ = 1/√(LC) → f₀ = 1/(2π√(LC))
At resonance, impedance is purely resistive (Z = R), and current is maximum (I = V/R).
Quality factor Q: Q = ω₀L/R = 1/(ω₀CR) = (1/R)√(L/C)
Q represents the sharpness of resonance. High Q → narrow bandwidth, high selectivity. Bandwidth: BW = f₀/Q = R/(2πL)
Q05
What is the time constant of a series RLC circuit?
▶
Answer
The time constant τ governs how quickly the transient decays. For an RLC circuit:
τ = 1/(ζωₙ) = 2L/R
The damping coefficient (Neper frequency): α = R/(2L) = ζωₙ
The transient dies out in approximately 5τ (5 time constants). For overdamped systems, two time constants exist: τ₁ = 1/|s₁| and τ₂ = 1/|s₂|. The larger time constant dominates the long-term behavior. For a simple RC circuit: τ = RC; for RL circuit: τ = L/R.
Q06
What is a CRO (Cathode Ray Oscilloscope)? How is it used to observe transient response?
▶
Answer
A CRO (Cathode Ray Oscilloscope) is an electronic test instrument that displays signal voltage as a function of time on its screen. To observe RLC transient response:
  • Apply a square wave input (which provides periodic step excitation) to the RLC circuit
  • Connect the output voltage across the capacitor or resistor to the CRO probe
  • Adjust time/div (timebase) to display the transient portion
  • Adjust volts/div to get appropriate amplitude
  • Use trigger to synchronize the display
Square wave is preferred because it provides repeated step inputs, making the transient observable periodically.
Q07
How do you simulate the transient response of a series RLC circuit in MATLAB?
▶
Answer
R=10; L=0.1; C=100e-6;
wn = 1/sqrt(L*C);
zeta = R/(2*sqrt(L/C));
num = [wn^2];
den = [1, 2*zeta*wn, wn^2];
sys = tf(num, den);
step(sys);
Or using impulse(sys) for impulse response. The lsim(sys, u, t) function can simulate response to any arbitrary input u.
Q08
What effect does increasing resistance R have on the transient response of a series RLC circuit?
▶
Answer
Increasing R increases the damping ratio ζ = R/(2√(L/C)):
  • Larger R → larger ζ → more damping
  • At small R: underdamped (oscillatory) response
  • At R = 2√(L/C): critical damping (ζ = 1), fastest non-oscillatory response
  • At large R: overdamped (slow, exponential) response
Increasing R also reduces the Q factor and increases bandwidth. It dissipates more energy, reducing oscillation amplitude and frequency. The natural frequency ωₙ = 1/√(LC) is independent of R; only the damping changes.
Q09
What is damped natural frequency? How is it related to the natural frequency?
▶
Answer
The damped natural frequency ωd is the actual frequency of oscillation observed in the underdamped transient response:
ωd = ωₙ√(1 - ζ²)
It is always less than the natural frequency ωₙ. The underdamped response can be written as:
v(t) = A·e^(-ζωₙt)·sin(ωdt + φ)
As ζ → 0: ωd → ωₙ (undamped oscillation). As ζ → 1: ωd → 0 (critical damping, no oscillation). This frequency is directly observable on the CRO as the frequency of the ringing waveform.
Q10
Why is a square wave used as input to study transient response instead of a step input?
▶
Answer
A square wave is used because:
  • A true DC step can only trigger the transient once, making it hard to observe on a CRO without a storage feature
  • A square wave provides periodic step excitations — each rising and falling edge triggers a transient response, allowing the CRO to synchronize and repeatedly display the same transient
  • The frequency must be chosen low enough that the transient decays completely before the next edge — typically f << 1/(5τ)
Mathematically, each half-cycle of the square wave is equivalent to an independent step input to the circuit.
Q11
Define the critical resistance of a series RLC circuit.
▶
Answer
The critical resistance Rc is the value of R at which the circuit becomes critically damped (ζ = 1). Setting ζ = 1:
Rc = 2√(L/C)
Example: For L = 10 mH and C = 10 μF: Rc = 2√(0.01/10⁻⁵) = 2×31.62 = 63.25 Ω.
If R < Rc → underdamped; R = Rc → critically damped; R > Rc → overdamped. The critically damped condition is important in applications like galvanometers and measuring instruments where fast, non-oscillatory response is desired.
Q12
What is the significance of poles in determining the transient response?
▶
Answer
The poles of the system transfer function (roots of the denominator) determine the transient response:
  • Poles on real axis (left half plane): exponentially decaying terms
  • Complex conjugate poles (LHP): damped sinusoidal oscillation
  • Poles on imaginary axis: sustained sinusoidal oscillation (marginally stable)
  • Poles in right half plane: growing (unstable) response
For the series RLC circuit: poles at s = -ζωₙ ± jωd. The real part (-ζωₙ) controls the rate of decay; the imaginary part (ωd) gives the oscillation frequency.

EXPERIMENT 05 · CONTROL SYSTEMS
Time Response of Linear Systems — Step & Square Wave, 1st & 2nd Order, Type 0,1
12 Questions
+
Q01
What is a first-order system? Write its standard transfer function and step response.
▶
Answer
A first-order system has a single energy storage element and its transfer function has only one pole. Standard form: H(s) = K/(τs + 1) where K is DC gain and τ is the time constant.
Unit step response: y(t) = K(1 - e^(-t/τ))·u(t)
Characteristics:
  • No overshoot
  • Reaches 63.2% of final value at t = τ
  • Reaches 99.3% at t ≈ 5τ (settling time)
  • Rise time (10% to 90%): tr ≈ 2.2τ
Examples: RC circuit (τ=RC), RL circuit (τ=L/R), thermal systems.
Q02
What is a second-order system? Write its standard form and define its time-domain specifications.
▶
Answer
Standard second-order transfer function:
H(s) = ωₙ² / (s² + 2ζωₙs + ωₙ²)
Time-domain specifications (underdamped case):
  • Rise Time tr: Time to go from 0% to 100% of final value ≈ (π-φ)/ωd
  • Peak Time tp: Time to reach first peak = π/ωd
  • Maximum Overshoot Mp: Mp = e^(-πζ/√(1-ζ²)) × 100%
  • Settling Time ts: Time to remain within ±2% ≈ 4/(ζωₙ), within ±5% ≈ 3/(ζωₙ)
  • Steady-state error ess: Final value - desired value
Q03
What is the type of a control system? Define Type 0, Type 1, and Type 2 systems.
▶
Answer
The type of a closed-loop system refers to the number of open-loop poles at the origin (s=0) — i.e., the number of pure integrators in the forward path.
Type 0: No integrator in open loop. Finite steady-state error for step input. Zero error for impulse.
Type 1: One integrator (1/s). Zero steady-state error for step input. Finite error for ramp input.
Type 2: Two integrators (1/s²). Zero error for step and ramp. Finite error for parabolic input.
Open-loop TF: G(s) = K·(s+z₁)... / [sⁿ·(s+p₁)...] where n = system type.
Q04
What are error constants? Define position, velocity, and acceleration error constants.
▶
Answer
Static error constants quantify the steady-state error for standard inputs:
Position error constant: Kp = lim(s→0) G(s)
Velocity error constant: Kv = lim(s→0) s·G(s)
Acceleration error constant: Ka = lim(s→0) s²·G(s)
Steady-state errors: ess(step) = 1/(1+Kp), ess(ramp) = 1/Kv, ess(parabola) = 1/Ka
Type 0: Kp finite, Kv=0, Ka=0. Type 1: Kp=∞, Kv finite, Ka=0. Type 2: Kp=∞, Kv=∞, Ka finite.
Q05
What is the effect of adding a zero or a pole to the transfer function on the step response?
▶
Answer
Adding a zero (in LHP): Increases overshoot and decreases rise time. The zero adds a derivative term, speeding up the response. Adding a zero at s = -z to a 2nd order system adds a derivative term (1 + s/z) to the numerator.
Adding a pole (in LHP): Slows down response — increases rise time, peak time, and settling time. It makes the system less responsive (low-pass filtering effect).
Adding a RHP zero: Causes undershoot (initial response in wrong direction) — characteristic of non-minimum phase systems.
Q06
What is the relationship between damping ratio ζ and percentage overshoot?
▶
Answer
For a second-order underdamped system, the percentage overshoot (Mp%) depends only on ζ:
Mp% = exp(-πζ/√(1-ζ²)) × 100%
Examples:
  • ζ = 0.1 → Mp ≈ 73%
  • ζ = 0.5 → Mp ≈ 16.3%
  • ζ = 0.707 → Mp ≈ 4.3%
  • ζ = 1.0 → Mp = 0% (critically damped)
Inversely: ζ = -ln(Mp/100) / √(π² + [ln(Mp/100)]²). Overshoot occurs only for ζ < 1 (underdamped).
Q07
What is a closed-loop transfer function? How does it differ from open-loop?
▶
Answer
For a unity feedback system with forward path G(s) and feedback H(s):
Open-loop TF: G(s)·H(s) — no feedback path closed
Closed-loop TF: T(s) = G(s)/(1 + G(s)H(s)) (for unity feedback, H=1: T = G/(1+G))
Open-loop control: simpler but sensitive to disturbances. Closed-loop (feedback) control: reduces steady-state error, reduces sensitivity to parameter variations, improves disturbance rejection, but may cause instability if not designed carefully.
Q08
How is the step response simulated in MATLAB for a given transfer function?
▶
Answer
num = [wn^2];
den = [1, 2*zeta*wn, wn^2];
sys = tf(num, den);
t = 0:0.01:10;
step(sys); % Auto-plots step response
% OR for more control:
[y, t] = step(sys, t);
plot(t, y); grid on;
For square wave response: u = square(2*pi*f*t);
lsim(sys, u, t);
Q09
What is the significance of the dominant pole approximation?
▶
Answer
In a higher-order system, poles closest to the imaginary axis (with smallest real parts) decay most slowly and dominate the transient response. Poles far in the LHP decay quickly and contribute little to the long-term behavior.
Dominant pole approximation: If a pair of complex poles is significantly closer to the jω-axis than other poles (by a factor of 5 or more), the system can be approximated as a second-order system using just those two poles. This simplifies analysis greatly. The approximation holds when non-dominant poles are at least 5× farther from jω-axis.
Q10
What is bandwidth? How is it related to rise time?
▶
Answer
Bandwidth (BW) of a control system is the frequency range over which the closed-loop frequency response is within 3 dB of its DC value (i.e., magnitude ≥ 0.707 of DC gain).
Relationship with rise time: For a second-order system:
BW ≈ 1/tr (approximately, inversely proportional)
More precisely: BW = ωₙ√(1-2ζ² + √(4ζ⁴ - 4ζ² + 2))
Larger bandwidth → faster rise time → faster system response. However, larger BW also amplifies high-frequency noise. There is a design tradeoff between speed (BW) and noise sensitivity.
Q11
Compare the step response of Type 0 and Type 1 systems.
▶
Answer
Type 0 system:
  • Closed-loop step response has a finite steady-state error
  • ess = 1/(1+Kp) where Kp = lim(s→0) G(s)
  • Higher gain K reduces error but may cause instability
Type 1 system:
  • Closed-loop step response has zero steady-state error
  • Kp = ∞, so ess(step) = 0
  • However, ramp input gives finite error = 1/Kv
The integrator in Type 1 systems acts as an error eliminator for constant inputs. PID controllers add integral action to increase system type and eliminate steady-state error.
Q12
What is the final value theorem? Apply it to find the steady-state value.
▶
Answer
The Final Value Theorem (FVT) states:
lim(t→∞) y(t) = lim(s→0) s·Y(s)
Conditions: All poles of s·Y(s) must be in the LHP (system must be stable).
Application: For a unit step input (1/s) to a system with TF G(s):
Y(s) = G(s)/s
y(∞) = lim(s→0) s · G(s)/s = G(0)
Example: G(s) = 10/(s+5) → y(∞) = 10/5 = 2. Similarly, the Initial Value Theorem: y(0⁺) = lim(s→∞) s·Y(s)

EXPERIMENT 06 · TWO-PORT NETWORKS
Determine Z and Y Parameters of the Given Two-Port Network
10 Questions
+
Q01
What is a two-port network? What are the port conditions?
▶
Answer
A two-port network is an electrical network with two pairs of terminals (port 1 and port 2) through which it interacts with external circuits. Variables: V₁, I₁ at port 1; V₂, I₂ at port 2 (I₂ is defined entering port 2 for Z/Y, leaving for ABCD).
Port conditions: The current entering the top terminal must equal the current leaving the bottom terminal at each port. These conditions must be satisfied for two-port parameters to be valid. The network may contain R, L, C elements, and dependent sources (but no independent sources for linear two-port analysis).
Q02
Define Z-parameters (impedance parameters). Write the defining equations.
▶
Answer
Z-parameters (open-circuit impedance parameters) express voltages in terms of currents:
V₁ = Z₁₁·I₁ + Z₁₂·I₂
V₂ = Z₂₁·I₁ + Z₂₂·I₂
Determination:
Z₁₁ = V₁/I₁ |I₂=0 (input impedance, port 2 open)
Z₁₂ = V₁/I₂ |I₁=0 (reverse transfer impedance, port 1 open)
Z₂₁ = V₂/I₁ |I₂=0 (forward transfer impedance, port 2 open)
Z₂₂ = V₂/I₂ |I₁=0 (output impedance, port 1 open)
Units: ohms (Ω). Measured with ports open-circuited.
Q03
Define Y-parameters (admittance parameters). Write the defining equations.
▶
Answer
Y-parameters (short-circuit admittance parameters) express currents in terms of voltages:
I₁ = Y₁₁·V₁ + Y₁₂·V₂
I₂ = Y₂₁·V₁ + Y₂₂·V₂
Determination:
Y₁₁ = I₁/V₁ |V₂=0 (input admittance, port 2 short-circuited)
Y₁₂ = I₁/V₂ |V₁=0 (reverse transfer admittance, port 1 short)
Y₂₁ = I₂/V₁ |V₂=0 (forward transfer admittance, port 2 short)
Y₂₂ = I₂/V₂ |V₁=0 (output admittance, port 1 short)
Units: siemens (S). Measured with ports short-circuited.
Q04
What is the relationship between Z and Y parameters?
▶
Answer
The Y-parameter matrix is the inverse of the Z-parameter matrix (and vice versa), provided the inverse exists:
[Y] = [Z]⁻¹
Explicitly: if ΔZ = Z₁₁Z₂₂ - Z₁₂Z₂₁ (determinant of Z matrix):
Y₁₁ = Z₂₂/ΔZ
Y₁₂ = -Z₁₂/ΔZ
Y₂₁ = -Z₂₁/ΔZ
Y₂₂ = Z₁₁/ΔZ
Note: Z parameters may not exist (e.g., if a port is short-circuited by a short circuit in the network), similarly Y parameters may not exist for certain networks. In those cases, alternative parameters (ABCD, h) must be used.
Q05
What are the conditions for reciprocity and symmetry in terms of Z and Y parameters?
▶
Answer
Reciprocity: A network containing only passive bilateral elements (R, L, C) is reciprocal. Condition:
Z₁₂ = Z₂₁ (for Z parameters)
Y₁₂ = Y₂₁ (for Y parameters)
Symmetry: The network is electrically identical when ports are interchanged. Condition:
Z₁₁ = Z₂₂ (for Z parameters)
Y₁₁ = Y₂₂ (for Y parameters)
A network can be reciprocal but not symmetric (e.g., L-network), or both reciprocal and symmetric (e.g., T or π networks with equal arms).
Q06
What is the π (pi) equivalent circuit of a two-port network in terms of Y parameters?
▶
Answer
Any two-port network can be represented by an equivalent π (pi) circuit using Y parameters:
  • Shunt admittance at port 1: Ya = Y₁₁ + Y₁₂
  • Shunt admittance at port 2: Yb = Y₂₂ + Y₁₂
  • Series (bridging) admittance: Yc = -Y₁₂
For a reciprocal network (Y₁₂=Y₂₁), this gives a straightforward three-element equivalent. The π equivalent is useful for modeling transmission lines, amplifiers, and filter sections.
Q07
What is the T equivalent circuit of a two-port network in terms of Z parameters?
▶
Answer
Any two-port network can be represented by a T (star) equivalent circuit using Z parameters:
  • Series arm at port 1: Za = Z₁₁ - Z₁₂
  • Series arm at port 2: Zb = Z₂₂ - Z₁₂
  • Shunt (common) arm: Zc = Z₁₂
For a reciprocal network (Z₁₂=Z₂₁), all three elements are determined. The T equivalent is used for modeling transmission lines, audio transformers, and RF matching networks.
Q08
When do Z-parameters or Y-parameters not exist for a two-port network?
▶
Answer
Z parameters do not exist when the open-circuit condition cannot yield a finite voltage — e.g., if the network contains a short circuit between port terminals, or if the Z matrix is singular (determinant = 0). Example: a simple series capacitor (shunt path at short-circuit ports becomes undefined).
Y parameters do not exist when the short-circuit condition is invalid — e.g., if the network contains a series inductor only with no path for current when ports are shorted (open circuit in the network). In such cases, hybrid (h, g) or ABCD parameters are used instead.
Q09
How do you experimentally measure Z₁₁ and Z₂₁?
▶
Answer
To measure Z₁₁ and Z₂₁ (with port 2 open-circuited, I₂ = 0):
  1. Leave port 2 open (no connection)
  2. Apply known voltage V₁ (or current I₁) to port 1
  3. Measure current I₁ entering port 1
  4. Z₁₁ = V₁/I₁
  5. Measure voltage V₂ appearing at open port 2 (with voltmeter)
  6. Z₂₁ = V₂/I₁
For AC circuits, use phasor measurements (magnitudes and phases). Repeat with port 1 open to find Z₁₂ and Z₂₂.
Q10
What are the practical applications of Z and Y parameters?
▶
Answer
Z parameters used in: Analysis of series-series interconnected networks, antenna impedance matching, transmission line modeling, characterizing RF/microwave networks.
Y parameters used in: Parallel-parallel interconnected networks, nodal analysis (Y-bus matrix in power systems), MOSFET and BJT small-signal modeling, filter design (ladder networks).
Both are used in: Network synthesis, equivalent circuit derivation, S-parameter conversion (for high-frequency/RF work), and simulation tools like SPICE and Agilent ADS.

EXPERIMENT 07 · TWO-PORT NETWORKS
Determine ABCD (Transmission) Parameters of the Two-Port Network
9 Questions
+
Q01
Define ABCD parameters. Write the defining equations and the physical significance of each parameter.
▶
Answer
ABCD (transmission/chain) parameters relate port 1 variables to port 2 variables (note: I₂ is defined as leaving port 2):
V₁ = A·V₂ - B·I₂
I₁ = C·V₂ - D·I₂
Physical significance:
A = V₁/V₂ |I₂=0 — Voltage ratio (port 2 open), dimensionless
B = V₁/(-I₂) |V₂=0 — Transfer impedance (port 2 short), units Ω
C = I₁/V₂ |I₂=0 — Transfer admittance (port 2 open), units S
D = I₁/(-I₂) |V₂=0 — Current ratio (port 2 short), dimensionless
Q02
What is the condition for reciprocity and symmetry in terms of ABCD parameters?
▶
Answer
Reciprocity: AD - BC = 1 (determinant of ABCD matrix = 1)
Symmetry: A = D
Passive lossless network: A and D are real, B and C are imaginary (for purely reactive networks).
For a reciprocal symmetric network: AD - BC = 1 and A = D. These conditions can be used to verify experimental measurements. If AD - BC ≠ 1 experimentally, it indicates measurement error or active/non-bilateral elements.
Q03
Why are ABCD parameters preferred for cascade networks?
▶
Answer
For a cascade (series chain) of N two-port networks, the overall ABCD matrix is simply the matrix product of the individual ABCD matrices:
[ABCD]overall = [ABCD]₁ · [ABCD]₂ · ... · [ABCD]N
This is extremely convenient because matrix multiplication is straightforward. In contrast, finding overall Z or Y parameters of cascaded networks requires complex conversions. ABCD parameters are thus the natural choice for transmission lines, filter ladders, attenuators, and communication networks where multiple two-ports are cascaded.
Q04
Derive ABCD parameters for a simple series impedance Z (two-port).
▶
Answer
For a series impedance Z (T-network with Z in series, nothing in shunt):
Using KVL: V₁ = V₂ + Z·I₁, and since there's no shunt element: I₁ = I₂ (current through is the same, with I₂ out = I₁ in).
Wait — with I₂ defined leaving: I₁ = -(-I₂) = I₂
V₁ = V₂ - Z·(-I₂) = V₂ + Z·I₂ ...
Actually: V₁ = 1·V₂ + Z·(-I₂) convention → A = 1, B = Z, C = 0, D = 1
ABCD matrix: [[1, Z], [0, 1]]
Check: AD - BC = 1·1 - Z·0 = 1 ✓ (reciprocal)
Q05
Derive ABCD parameters for a shunt admittance Y (two-port).
▶
Answer
For a shunt admittance Y across the ports (both ports share the same voltage): V₁ = V₂ (no series element).
Current: I₁ = YV₁ + (-I₂) = YV₂ - I₂
Comparing with I₁ = C·V₂ - D·I₂:
A = 1, B = 0, C = Y, D = 1
ABCD matrix: [[1, 0], [Y, 1]]
Check: AD - BC = 1·1 - 0·Y = 1 ✓
These two building blocks (series Z and shunt Y) can be cascaded to build any ladder network's overall ABCD matrix by matrix multiplication.
Q06
How do you convert ABCD parameters to Z parameters?
▶
Answer
Conversion from ABCD to Z parameters (assuming C ≠ 0):
Z₁₁ = A/C
Z₁₂ = (AD-BC)/C = 1/C (for reciprocal network where AD-BC=1)
Z₂₁ = 1/C
Z₂₂ = D/C
Conversion from ABCD to Y parameters (assuming B ≠ 0):
Y₁₁ = D/B, Y₁₂ = -(AD-BC)/B = -1/B
Y₂₁ = -1/B, Y₂₂ = A/B
Q07
What are the ABCD parameters of an ideal transformer with turns ratio n:1?
▶
Answer
For an ideal transformer with turns ratio n:1 (V₁/V₂ = n, I₁/I₂ = 1/n):
V₁ = n·V₂ → A = n, B = 0
I₁ = (1/n)·I₂ → C = 0, D = 1/n
ABCD matrix: [[n, 0], [0, 1/n]]
Check: AD - BC = n·(1/n) - 0 = 1 ✓
Note: A = D only when n = 1/n, i.e., n = 1 (unity ratio), so a transformer is symmetric only for n = 1. The ideal transformer is lossless and reciprocal.
Q08
What is the input impedance of a two-port network terminated in load ZL, in terms of ABCD parameters?
▶
Answer
When port 2 is loaded with impedance ZL (V₂ = -I₂·ZL, with I₂ leaving port 2):
Zin = V₁/I₁ = (A·ZL + B)/(C·ZL + D)
This is very useful in transmission line analysis, filter design, and impedance transformation. Special cases:
  • Open circuit (ZL = ∞): Zin = A/C
  • Short circuit (ZL = 0): Zin = B/D
  • Characteristic impedance: Zin = Z₀ when Z₀ = √(B/C) (for symmetric network A=D)
Q09
What are the practical applications of ABCD parameters?
▶
Answer
Applications of ABCD parameters:
  • Transmission lines: Characterizing long-distance power and signal transmission
  • Filter design: Ladder networks (LC filters) easily analyzed by cascading sections
  • Power systems: ABCD constants of transmission lines (A,B,C,D relate sending and receiving end quantities)
  • Microwave engineering: Modeling waveguide sections and discontinuities
  • Optical systems: Ray transfer matrices (ABCD matrices) for lens and mirror systems
  • Attenuator and amplifier chains: Overall gain and impedance easily computed

EXPERIMENT 08 · CIRCUIT THEOREMS
Verify Various Theorems in AC Circuits
11 Questions
+
Q01
State and explain Thevenin's theorem. How is it applied to AC circuits?
▶
Answer
Thevenin's Theorem: Any linear two-terminal network containing sources and impedances can be replaced by a single voltage source Vth in series with an impedance Zth.
Finding Vth: Remove load, find open-circuit voltage across terminals.
Finding Zth: Deactivate all independent sources (short V-sources, open I-sources), find impedance seen from terminals.
In AC circuits: All quantities are phasors. Vth is the open-circuit phasor voltage; Zth is the complex impedance (has both real and imaginary parts). The theorem simplifies analysis when multiple loads need to be analyzed for the same source network.
Q02
State Norton's theorem and explain its relationship to Thevenin's theorem.
▶
Answer
Norton's Theorem: Any linear two-terminal network can be replaced by a current source IN in parallel with impedance ZN.
Finding IN: Short-circuit the load terminals and find the current through the short (Norton current).
Finding ZN: Same as Zth (ZN = Zth).
Relationship: Norton and Thevenin are duals:
Vth = IN × ZN
ZN = Zth
Source transformation converts between them. Both theorems yield identical results when connected to the same load impedance ZL.
Q03
State the Maximum Power Transfer theorem for AC circuits. What is the condition for maximum power?
▶
Answer
Maximum Power Transfer Theorem: Maximum power is delivered to a load when the load impedance ZL is the complex conjugate of the Thevenin source impedance Zth.
Condition: ZL = Z*th = Rth - jXth
Under this condition: Load resistance RL = Rth and Load reactance XL = -Xth (cancels source reactance).
Maximum power: Pmax = |Vth|²/(4Rth)
For purely resistive circuits: ZL = Zth → RL = Rth (load = source resistance). The efficiency at maximum power is 50% (equal power in source and load).
Q04
State the Superposition theorem. When can it be applied?
▶
Answer
Superposition Theorem: In a linear circuit with multiple independent sources, the total response (voltage or current) at any element is the algebraic sum of individual responses due to each independent source acting alone, with all other sources deactivated.
Conditions for application:
  • Circuit must be linear (obeys superposition)
  • Cannot be applied directly to power (P ≠ P₁ + P₂) as power is non-linear (I²R)
  • Dependent sources are NOT deactivated
In AC circuits: Add phasor contributions from each source separately.
Q05
State the Reciprocity theorem. Give an example.
▶
Answer
Reciprocity Theorem: In a linear bilateral (passive) network with a single excitation, if we interchange the position of the source and the response (ammeter/voltmeter), the magnitude of the response remains unchanged.
Conditions: Only one independent source; network must be linear and bilateral (no dependent sources, no active elements).
Example: If a voltage source V at node A produces current I at branch B, then the same source V placed at branch B produces the same current I at node A.
This is related to Z₁₂ = Z₂₁ and Y₁₂ = Y₂₁ for two-port networks.
Q06
State Millman's theorem. Where is it useful?
▶
Answer
Millman's Theorem: For multiple voltage sources connected in parallel (through their series impedances) to a common load, the equivalent voltage and impedance are:
Veq = (V₁Y₁ + V₂Y₂ + ... + VₙYₙ) / (Y₁ + Y₂ + ... + Yₙ)
Zeq = 1/(Y₁ + Y₂ + ... + Yₙ)
where Yᵢ = 1/Zᵢ.
Useful for: Parallel generator systems, power grid analysis with multiple sources, finding common bus voltage when multiple branches with EMF sources are connected in parallel. Simplifies circuits that would otherwise require extensive mesh/nodal analysis.
Q07
State Tellegen's theorem. What does it express?
▶
Answer
Tellegen's Theorem: For any network satisfying KVL and KCL, the sum of instantaneous power delivered to all branches is zero:
Σ vₖ·iₖ = 0 (summed over all branches k)
This is essentially a statement of energy conservation. It applies to any lumped network regardless of the nature of elements (linear, nonlinear, time-varying). It can be applied between two different networks with the same topology (not necessarily the same elements).
Significance: Provides a powerful tool for deriving other network theorems and proving network properties. It is more general than other circuit theorems.
Q08
How do you experimentally verify Thevenin's theorem?
▶
Answer
Step 1 — Original circuit: Connect load RL to circuit. Measure voltage VL and current IL across/through RL.
Step 2 — Find Vth: Remove RL. Measure open-circuit voltage Voc = Vth at terminals using voltmeter.
Step 3 — Find Rth/Zth: Deactivate sources (short V-sources, open I-sources). Measure resistance/impedance between terminals with LCR meter or calculate from component values.
Step 4 — Verify: Build Thevenin equivalent (Vth in series with Zth + RL). Calculate and measure VL = Vth·RL/(Zth+RL). Compare with Step 1. If they match ✓, Thevenin's theorem is verified.
Q09
What is the compensation theorem?
▶
Answer
Compensation Theorem: Any element Z in a network can be replaced by a voltage source of value V = Z·I (where I is the current through Z) with zero impedance (short circuit) for the branch, and the currents and voltages throughout the rest of the network remain unchanged.
Application: Used to find changes in network response when an element value changes by ΔZ. The change ΔZ can be compensated by a source of value ΔV = ΔZ·I. This allows incremental analysis without solving the full circuit again, useful in sensitivity analysis and network design.
Q10
How does the presence of dependent sources affect the application of circuit theorems?
▶
Answer
Dependent (controlled) sources complicate theorem application:
  • Thevenin/Norton: Dependent sources are NOT deactivated when finding Zth. Must use a test voltage/current method: apply Vtest at terminals, find Itest (or vice versa), then Zth = Vtest/Itest. Zth may be negative with dependent sources.
  • Superposition: Dependent sources remain active; only independent sources are turned off one at a time.
  • Reciprocity: Does NOT apply to networks with dependent sources.
  • Maximum Power Transfer: Still applies, but Zth calculation must include dependent sources.
Q11
What is the difference between ideal and practical voltage and current sources?
▶
Answer
Ideal Voltage Source: Maintains constant terminal voltage regardless of current drawn. Internal impedance = 0.
Practical Voltage Source: Has internal series impedance Zs. Terminal voltage drops under load: V = Vs - I·Zs.
Ideal Current Source: Maintains constant current regardless of terminal voltage. Internal impedance = ∞.
Practical Current Source: Has internal shunt impedance Zp. Some current is diverted internally.
In Thevenin/Norton analysis: ideal V-sources are shorted (internal Z=0) and ideal I-sources are opened (internal Z=∞) when finding Thevenin impedance.

EXPERIMENT 09 · TWO-PORT NETWORKS
Determine Hybrid (h) Parameters of the Given Two-Port Network
9 Questions
+
Q01
Define hybrid h-parameters. Write the defining equations and physical significance of each.
▶
Answer
Hybrid parameters use mixed variables (V₁, I₂ as independent, I₁, V₂ as dependent):
V₁ = h₁₁·I₁ + h₁₂·V₂
I₂ = h₂₁·I₁ + h₂₂·V₂
Physical significance:
h₁₁ = V₁/I₁|V₂=0 — Short-circuit input impedance (Ω)
h₁₂ = V₁/V₂|I₁=0 — Open-circuit reverse voltage gain (dimensionless)
h₂₁ = I₂/I₁|V₂=0 — Short-circuit forward current gain (dimensionless)
h₂₂ = I₂/V₂|I₁=0 — Open-circuit output admittance (S)
Each parameter has different units — hence "hybrid"!
Q02
Why are h-parameters particularly important in transistor circuit analysis?
▶
Answer
H-parameters are ideal for BJT (Bipolar Junction Transistor) analysis because:
  • The transistor naturally has a low input impedance and high output impedance, matching the h-parameter model
  • h₂₁ = hfe is the current gain β of the transistor (most important parameter)
  • Input port is naturally short-circuit tested (base-emitter junction); output port is open-circuit tested (collector)
  • Manufacturers specify BJT as hie, hfe, hre, hoe for CE, CB, CC configurations
The h-parameter model is the standard small-signal model used in amplifier design and analysis.
Q03
What is the condition for reciprocity in h-parameters?
▶
Answer
For a reciprocal (passive bilateral) two-port network, the h-parameters satisfy:
h₁₂ = -h₂₁
Note the negative sign (unlike Z and Y parameters where the cross parameters are simply equal). This is because the h-matrix for a reciprocal network has det[h] = h₁₁h₂₂ - h₁₂h₂₁ = 1... actually for a reciprocal network: Δh = h₁₁h₂₂ - h₁₂h₂₁ = 1 is not the condition. The reciprocity condition is simply h₁₂ = -h₂₁.
Symmetry condition: Δh = h₁₁h₂₂ - h₁₂h₂₁ = 1
Q04
How do you convert h-parameters to Z-parameters?
▶
Answer
Let Δh = h₁₁h₂₂ - h₁₂h₂₁ (determinant of h matrix). Conversion to Z:
Z₁₁ = Δh/h₂₂
Z₁₂ = h₁₂/h₂₂
Z₂₁ = -h₂₁/h₂₂
Z₂₂ = 1/h₂₂
Conversion to Y (g-parameters inverse of h): [g] = [h]⁻¹
g₁₁ = h₂₂/Δh, g₁₂ = -h₁₂/Δh, g₂₁ = -h₂₁/Δh, g₂₂ = h₁₁/Δh
Q05
What are g-parameters (inverse hybrid)? When are they preferred over h-parameters?
▶
Answer
G-parameters (inverse hybrid or admittance-impedance parameters) use I₁ and V₂ as independent variables:
I₁ = g₁₁V₁ + g₁₂I₂
V₂ = g₂₁V₁ + g₂₂I₂
[g] = [h]⁻¹
G-parameters preferred when:
  • Network has high input impedance and low output impedance (opposite of BJT — more suitable for FETs/MOSFETs)
  • Input port is open-circuit tested, output is short-circuit tested
  • g₂₁ represents voltage gain (FET parameter gm relates to this)
For series-parallel network interconnection.
Q06
How do you experimentally determine h-parameters?
▶
Answer
Step 1 — Short-circuit port 2 (V₂ = 0):
Apply known I₁, measure V₁ → h₁₁ = V₁/I₁
Measure I₂ → h₂₁ = I₂/I₁
Step 2 — Open-circuit port 1 (I₁ = 0):
Apply known V₂, measure V₁ → h₁₂ = V₁/V₂
Measure I₂ → h₂₂ = I₂/V₂
For AC circuits, use signal generator and oscilloscope/LCR meter. All values are complex (phasors).
Q07
What type of network interconnection uses h-parameters?
▶
Answer
H-parameters are associated with the series-parallel (SP) interconnection of two-port networks:
  • Port 1 inputs are connected in series (voltages add: V₁ = V₁a + V₁b, currents are equal)
  • Port 2 outputs are connected in parallel (currents add: I₂ = I₂a + I₂b, voltages are equal)
Overall h-matrix: [h] = [ha] + [hb]
This is why h-parameters of series-parallel interconnected networks are simply added — as will be verified in Experiment 13.
Q08
How are h-parameters of a BJT transistor in CE configuration defined (hie, hfe, hre, hoe)?
▶
Answer
For Common Emitter (CE) BJT configuration:
hie = h₁₁ — Input impedance (short-circuit) ≈ rπ (base-emitter resistance), ~1-2 kΩ
hre = h₁₂ — Reverse voltage ratio (open-circuit) ≈ 10⁻⁴ (very small, often neglected)
hfe = h₂₁ — Forward current gain (short-circuit) = β (current gain, ~50-300)
hoe = h₂₂ — Output admittance (open-circuit) ≈ 1/ro (output resistance), ~25 μS
These are specified on transistor datasheets and used for AC small-signal amplifier design.
Q09
Compare all six two-port parameter sets in terms of independent variables.
▶
Answer
Summary of two-port parameter sets:
Z (impedance): Independent I₁,I₂ → Dependent V₁,V₂. Measured open-circuit.
Y (admittance): Independent V₁,V₂ → Dependent I₁,I₂. Measured short-circuit.
h (hybrid): Independent I₁,V₂ → Dependent V₁,I₂. Mixed measurements.
g (inverse hybrid): Independent V₁,I₂ → Dependent I₁,V₂. Mixed measurements.
ABCD (transmission): Independent V₂,-I₂ → Dependent V₁,I₁. Port 2 to port 1.
A'B'C'D' (inverse transmission): Independent V₁,-I₁ → Dependent V₂,I₂. Port 1 to port 2.

Cascade Connection and ABCD Parameters of Two-Port Networks
8 Questions
+
Q01
What is a cascade (tandem) connection of two-port networks? State the property of ABCD matrices in cascade.
▶
Answer
In a cascade connection, the output port (port 2) of network Na is directly connected to the input port (port 1) of network Nb — i.e., V₂a = V₁b and I₂a = -I₁b (sign convention: current leaving Na enters Nb).
ABCD matrix property:
[ABCD]overall = [ABCD]a × [ABCD]b
This is a simple matrix multiplication. For n cascaded networks:
[T] = [T₁][T₂][T₃]...[Tₙ]
Note: Order matters (matrix multiplication is not commutative). Cascading is used in filters, amplifier stages, and transmission line sections.
Q02
What is the port condition that must be satisfied for cascade connection to be valid?
▶
Answer
The port condition for cascade: when network Nb is connected to Na, the current entering port 1 of Nb must equal the current leaving port 2 of Na. This requires that no current is "stolen" by external connections at the intermediate node.
Formally: I₂a = I₁b (with appropriate sign convention). If the intermediate connection draws current from external connections (violating port condition), the cascade formula [T] = [Ta][Tb] is invalid. This can be tested using the Brune test: short-circuit test the intermediate terminals and verify no current flows in the external circuit.
Q03
Compute the ABCD matrix of a cascade of a series impedance Z₁ followed by a shunt admittance Y₁.
▶
Answer
Series Z₁: T₁ = [[1, Z₁], [0, 1]]
Shunt Y₁: T₂ = [[1, 0], [Y₁, 1]]
Overall: T = T₁ × T₂ = [[1, Z₁], [0, 1]] × [[1, 0], [Y₁, 1]]
T = [[(1+Z₁Y₁), Z₁], [Y₁, 1]]
So A = 1+Z₁Y₁, B = Z₁, C = Y₁, D = 1. Verify: AD-BC = (1+Z₁Y₁)·1 - Z₁·Y₁ = 1 ✓ (reciprocal)
This is the L-section (low-pass) ladder network ABCD matrix.
Q04
What is the characteristic impedance of a symmetric two-port network?
▶
Answer
The characteristic impedance Z₀ of a symmetric two-port (where A = D) is defined as the impedance at port 1 when port 2 is terminated in Z₀ itself (image impedance = load impedance):
Z₀ = √(B/C) (for symmetric network A=D)
When terminated in Z₀, the input impedance also equals Z₀ — a property useful for impedance matching.
Propagation constant: γ = cosh⁻¹(A) = cosh⁻¹(D)
These concepts extend to transmission line theory where Z₀ is the characteristic impedance and γ is the propagation constant.
Q05
Give practical examples where cascade connection of two-ports is used.
▶
Answer
Practical examples of cascade connection:
  • Multistage amplifiers: Each amplifier stage is a two-port; cascade gives overall gain
  • Ladder filters (LC): Low-pass, high-pass, band-pass filters built from cascaded L-sections
  • Transmission lines: Long lines modeled as multiple cascaded π or T sections
  • Attenuator pads: Multiple cascaded attenuator stages
  • Optical systems: Ray tracing through lens systems using ABCD matrices
  • Signal processing chains: ADC + filter + DSP, each as a cascade two-port
Q06
What are image parameters? How are they related to ABCD parameters?
▶
Answer
Image parameters are defined such that when a network is terminated in its image impedances, the impedances seen looking into each port equal the respective terminations.
Image impedances:
Zi1 = √(AB/CD) (image impedance at port 1)
Zi2 = √(BD/AC) (image impedance at port 2)
Image propagation constant:
e^(-γ) = √(AD) - √(BC) = √(AD-BC) + ...
Actually: cosh(γ) = √(AD)
Image parameters are used in filter design theory (image parameter method) for designing ladder filters.
Q07
How do you find the overall Z parameters after cascade connection?
▶
Answer
For a cascade connection, Z parameters do not simply add (unlike series-series connection). The procedure is:
  1. Convert Z parameters of each network to ABCD: [T₁] and [T₂]
  2. Compute overall ABCD: [T] = [T₁] × [T₂]
  3. Convert overall ABCD back to Z parameters:
    Z₁₁ = A/C, Z₁₂ = Z₂₁ = 1/C (if reciprocal)
    Z₂₂ = D/C
This conversion approach is necessary because cascade connection fundamentally operates in ABCD domain.
Q08
What happens to ABCD parameters when a network is reversed (ports interchanged)?
▶
Answer
When a two-port network is reversed (port 1 ↔ port 2, signal flows from right to left), the new ABCD matrix [T'] is:
A' = D, B' = B, C' = C, D' = A
So the new matrix: T' = [[D, B], [C, A]]
For a symmetric network (A = D): T' = T (matrix is unchanged when reversed).
For a reciprocal network: det(T') = D'A' - B'C' = DA - BC = 1, which remains 1 as expected. This is useful when analyzing signal flow in the reverse direction.

Series-Series Connection and Z Parameters
7 Questions
+
Q01
What is a series-series (SS) connection of two two-port networks? State the Z-matrix property.
▶
Answer
In a series-series connection, both port 1 inputs are connected in series AND both port 2 outputs are connected in series:
  • Port 1: V₁ = V₁a + V₁b, I₁a = I₁b = I₁
  • Port 2: V₂ = V₂a + V₂b, I₂a = I₂b = I₂
Z-matrix property: The overall Z matrix is the sum of individual Z matrices:
[Z]overall = [Za] + [Zb]
This is the most convenient property — Z parameters simply add for series-series interconnection. This is analogous to series resistance addition in simple circuits.
Q02
What is the port condition (Brune's condition) for series-series connection?
▶
Answer
The port condition requires that the current entering port 1 terminal of each individual network equals the current leaving its port 1 return terminal — even after the interconnection is made.
Brune's test: Short-circuit both ports of the combined network. Check if any current flows through a wire connecting the input terminal common to both networks. If no such current flows, the port condition is satisfied and [Z]total = [Za] + [Zb]. If current does flow, an isolating transformer must be used.
Violation of port conditions leads to incorrect parameter addition.
Q03
Why does the series-series connection of two-ports result in Z parameter addition?
▶
Answer
The Z parameters define: V = Z·I. In series-series connection:
V₁ = V₁a + V₁b = Za₁₁I₁ + Za₁₂I₂ + Zb₁₁I₁ + Zb₁₂I₂ = (Za₁₁+Zb₁₁)I₁ + (Za₁₂+Zb₁₂)I₂
Similarly: V₂ = (Za₂₁+Zb₂₁)I₁ + (Za₂₂+Zb₂₂)I₂
Therefore Z₁₁ = Za₁₁+Zb₁₁, etc., giving [Z] = [Za]+[Zb]. The key is that series connection makes voltages add while currents remain equal — exactly what Z parameters describe (currents as independent variables).
Q04
Give practical examples of series-series interconnection.
▶
Answer
Practical examples:
  • Series feedback amplifiers: Voltage-series feedback (series at input and output) — overall Z-matrix = forward + feedback network Z-matrices
  • Differential voltage sensing: Two networks sensing different voltages in series
  • Transformer-coupled balanced circuits
  • Series impedance compensation: Adding a correction network in series with a transmission system
  • NMR/MRI resonant circuits where tuning networks are series-connected
Series-series is the topology of choice when dealing with voltage-controlled, current-input systems.
Q05
How do you experimentally verify the Z-parameter addition property of series-series connection?
▶
Answer
Procedure:
  1. Measure Z parameters of network Na individually: [Za]
  2. Measure Z parameters of network Nb individually: [Zb]
  3. Compute theoretical overall Z matrix: [Z]theory = [Za] + [Zb]
  4. Connect Na and Nb in series-series configuration
  5. Measure Z parameters of the combined network: [Z]measured
  6. Verify: [Z]measured ≈ [Z]theory
Check Brune's port condition before connecting. If port condition fails, results will not match — use isolating transformers to fix this.
Q06
What is the effect on input and output impedance of series-series feedback in amplifiers?
▶
Answer
For series-series feedback (voltage-series at input, current-sensing at output) in amplifiers:
  • Input impedance increases: Rin_f = Rin(1 + Aβ) where Aβ is the loop gain. Higher Rin is desirable for voltage amplifiers.
  • Output impedance increases: Rout_f = Rout(1 + Aβ). Higher Rout means the output behaves more like a current source.
  • Gain decreases: Af = A/(1 + Aβ) — but gain becomes more stable (desensitized to transistor parameter variations).
Q07
What parameter set would you use if the series-series port condition is not satisfied?
▶
Answer
If the port condition for series-series connection fails (currents through the interconnecting wires are non-zero), the simple addition [Z] = [Za]+[Zb] is invalid. In this case:
  • Use ABCD parameters (always valid for cascade) after connecting the networks in cascade instead
  • Use the full network analysis (nodal or mesh) by treating all network elements together
  • Use isolating transformers to enforce port conditions, then apply Z-addition
  • Apply graph theory (topological methods) for general interconnections
This is a common pitfall in two-port network analysis.

Parallel-Parallel Connection and Y Parameters
7 Questions
+
Q01
What is a parallel-parallel (PP) connection? State the Y-matrix addition property.
▶
Answer
In a parallel-parallel connection, both input ports are in parallel AND both output ports are in parallel:
  • Port 1: V₁a = V₁b = V₁ (equal voltages); I₁ = I₁a + I₁b (currents add)
  • Port 2: V₂a = V₂b = V₂ (equal voltages); I₂ = I₂a + I₂b (currents add)
Y-matrix property:
[Y]overall = [Ya] + [Yb]
Y parameters add in parallel-parallel connection. This is the dual of Z addition in series-series connection, and analogous to parallel admittance addition in simple circuits.
Q02
Why do Y parameters add in parallel-parallel connection?
▶
Answer
Y parameters define: I = Y·V. In parallel-parallel connection, voltages are equal at both ports and currents add:
I₁ = I₁a + I₁b = Ya₁₁V₁ + Ya₁₂V₂ + Yb₁₁V₁ + Yb₁₂V₂ = (Ya₁₁+Yb₁₁)V₁ + (Ya₁₂+Yb₁₂)V₂
Similarly for I₂, giving [Y] = [Ya]+[Yb]. The parallel connection makes voltages equal and currents add — exactly the setup where Y parameters (voltages as independent variables) are naturally applicable. This mirrors how parallel admittances simply add (Y = Y₁ + Y₂) in basic circuit theory.
Q03
What is the port condition for parallel-parallel connection? What happens if it's violated?
▶
Answer
The port condition for PP connection requires that when both networks are connected in parallel, the individual port currents maintain their original values (port currents are not redistributed by common ground loops).
Testing: apply voltages V₁ = 0, V₂ = V at the combined network. Check if any circulating current appears between the ground terminals of Na and Nb at port 1. If circulating current exists, port condition is violated.
Consequence of violation: [Y]measured ≠ [Ya]+[Yb]. Must use isolating transformers or proceed with full circuit analysis (node voltage method treating all elements together).
Q04
Give practical examples of parallel-parallel interconnected two-port networks.
▶
Answer
Practical examples:
  • Shunt-shunt feedback amplifiers: Current-sensing at input and output — parallel feedback network added to amplifier, total Y = amplifier Y + feedback Y
  • Parallel combination of filter sections: Two filter branches connected in parallel
  • Power system Y-bus matrix: Parallel connection of transmission lines represented by admittances
  • Parallel resonant circuits with additional loading
  • Differential amplifier with common-mode feedback
Parallel-parallel is preferred when voltage is the controlled variable and current is the output.
Q05
What is the effect of parallel-parallel feedback on amplifier input and output impedance?
▶
Answer
For shunt-shunt (parallel-parallel) feedback amplifiers:
  • Input impedance decreases: Rin_f = Rin/(1 + Aβ). Lower Rin makes the input more like a current receiver — desirable for transimpedance amplifiers.
  • Output impedance decreases: Rout_f = Rout/(1 + Aβ). Lower Rout means the output behaves more like an ideal voltage source.
  • Gain decreases but becomes more stable.
This contrasts with series-series feedback where impedances increase. The type of feedback topology determines which impedances are increased or decreased.
Q06
How is the Y-bus matrix in power systems related to Y-parameters of two-port networks?
▶
Answer
The Y-bus matrix (admittance bus matrix) in power systems is a generalization of two-port Y-parameters to n-port networks. For a power network with n buses:I = Ybus · V
where I is the vector of injected currents and V is the vector of bus voltages.
Each transmission line between buses i and j is a two-port with admittance yij, and its contribution to Ybus follows Y-parameter parallel-connection addition:
  • Diagonal: Yii = sum of all admittances connected to bus i
  • Off-diagonal: Yij = -yij (negative of admittance between buses i and j)
This is exactly the parallel admittance addition principle of Y-parameters applied to multi-port networks.
Q07
Compare the four types of interconnections (SS, PP, SP, PS) and their associated parameters.
▶
Answer
Summary of two-port interconnections:
Series-Series (SS): Port 1 series, Port 2 series → [Z] = [Za]+[Zb]
Parallel-Parallel (PP): Port 1 parallel, Port 2 parallel → [Y] = [Ya]+[Yb]
Series-Parallel (SP): Port 1 series, Port 2 parallel → [h] = [ha]+[hb]
Parallel-Series (PS): Port 1 parallel, Port 2 series → [g] = [ga]+[gb]
Cascade (Tandem): Port 2 of one → Port 1 of next → [T] = [Ta]×[Tb]

Series-Parallel Connection and h Parameters
8 Questions
+
Q01
What is a series-parallel (SP) connection? State the h-matrix addition property.
▶
Answer
In a series-parallel connection:
  • Port 1 — Series: V₁ = V₁a + V₁b (voltages add), I₁a = I₁b = I₁ (currents equal)
  • Port 2 — Parallel: V₂a = V₂b = V₂ (voltages equal), I₂ = I₂a + I₂b (currents add)
h-matrix property:
[h]overall = [ha] + [hb]
H-parameters add in series-parallel interconnection. This makes sense because h-parameters use I₁ and V₂ as independent variables — matching the series (equal current) at port 1 and parallel (equal voltage) at port 2 conditions.
Q02
Why do h-parameters add in series-parallel connection? Derive the result.
▶
Answer
H-parameter equations: V₁ = h₁₁I₁ + h₁₂V₂, I₂ = h₂₁I₁ + h₂₂V₂
For SP connection: I₁ is common to both, V₂ is common to both, V₁ and I₂ are summed:
V₁ = V₁a+V₁b = (ha₁₁+hb₁₁)I₁ + (ha₁₂+hb₁₂)V₂
I₂ = I₂a+I₂b = (ha₂₁+hb₂₁)I₁ + (ha₂₂+hb₂₂)V₂
Therefore: h₁₁ = ha₁₁+hb₁₁, h₁₂ = ha₁₂+hb₁₂
h₂₁ = ha₂₁+hb₂₁, h₂₂ = ha₂₂+hb₂₂
giving [h] = [ha]+[hb] ∎
Q03
What is the port condition for series-parallel connection?
▶
Answer
For the SP connection to be valid:
  • At port 1 (series): The current flowing into the upper terminal of each sub-network must equal the current flowing out of its lower terminal. No current should leak through parasitic paths created by the interconnection.
  • At port 2 (parallel): Connecting port 2 of Na in parallel with port 2 of Nb must not create ground loops that alter the individual port currents.
Testing: Insert voltage source V between the ground terminals at port 2. If no current flows, the condition is satisfied. Use isolating transformers if needed to enforce port conditions.
Q04
Give the example of series-parallel feedback in amplifiers.
▶
Answer
A classic example of SP (voltage-series, current-shunt) feedback is the emitter follower (common collector) BJT amplifier:
  • Series at input: Feedback voltage (across emitter resistor) is in series with input — senses output voltage
  • Parallel at output: Output current is sampled by the shunt feedback element
Effect: input impedance increases, output impedance decreases, voltage gain ≈ 1 (unity gain buffer). Another example: op-amp non-inverting voltage follower with 100% feedback — R₁ = ∞ configuration.
Q05
Compare series-parallel (SP) and parallel-series (PS) connections.
▶
Answer
Series-Parallel (SP):
  • Port 1: series (voltages add, currents equal)
  • Port 2: parallel (currents add, voltages equal)
  • Associated parameter: h-parameters ([h] = [ha]+[hb])
  • Feedback type: Series input, shunt output (voltage-series feedback)
Parallel-Series (PS):
  • Port 1: parallel (currents add, voltages equal)
  • Port 2: series (voltages add, currents equal)
  • Associated parameter: g-parameters ([g] = [ga]+[gb])
  • Feedback type: Shunt input, series output (current-shunt feedback)
SP and PS are duals of each other.
Q06
How does series-parallel feedback affect transimpedance (V/I) of an amplifier?
▶
Answer
Series-parallel feedback is used in transresistance amplifiers (current-to-voltage converters):
  • Input: current I_in injected at node (shunt at input) — actually this is shunt-series. For SP: voltage in, voltage out (voltage gain amplifier)
  • SP feedback stabilizes voltage gain Avf = Vo/Vi
  • Avf = Av/(1 + Avβ) where β = feedback fraction (voltage divider)
  • Input impedance: increases by factor (1 + Avβ)
  • Output impedance: decreases by factor (1 + Avβ)
Op-amp non-inverting amplifier with resistive voltage divider is the canonical SP feedback circuit.
